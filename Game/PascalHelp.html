Помощь по языку Pascal

Паска́ль (англ. Pascal) — один из наиболее известных языков программирования, используется для обучения программированию в старших классах и на первых курсах вузов, является базой для ряда других языков.

Структура программы на языке Pascal

Любая программа на языке Паскаль начинается с ключевого слова Program и следующего за ним имени программы.

Следущие строчки являются разделом описания, здесь объявляются переменные с помощью ключевого слова Var, а так же константы с помощью Const.

После раздела описания следует тело программы, оно начинается с ключевого слова Begin и содержит все команды для выполнения компьютером, тело программы заканчивается ключевым словом End с точкой в конце.
